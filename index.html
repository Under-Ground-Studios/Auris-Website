About

Built for scripters

Auris is a lightweight editor and execution client for Roblox Lua. No bloat, just a clean workspace.

Monaco Editor

Syntax highlighting, tabbed editing, and a fast search-as-you-type switcher.

Script Hub

Browse and run community scripts directly from the editor. No copy-pasting.
