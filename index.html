What's inside

Everything sits in one editor

No separate launchers, no bloat. Auris is one window: an editor, a script hub, and a key you only have to think about once a day.

Monaco-based editor

The same editing engine behind VS Code — real syntax highlighting, multi-tab support, and a search-as-you-type tab switcher.

Script hub, built in

Pull scripts from the hub and run them without leaving the editor. Favorite the ones you use every session.

Themes that don't fight you

A dark interface by default, with the kind of contrast and spacing that stays comfortable after the tenth tab.

Getting started

Three steps, in order

This is the actual order things happen in — key first, then the download, then you're in the editor.

01 / KEY

Get a key

Head to the Key System page and generate one. It's free and valid for 24 hours.

02 / INSTALL

Download Auris

Grab the current build for Windows from the download page and run the installer.

03 / RUN

Paste your key and go

Drop your key into the prompt on first launch, open a tab, and start writing.
