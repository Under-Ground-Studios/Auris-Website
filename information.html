Information

What Auris actually does

The short version: it's an editor and a runner. The rest of this page is the long version.

Editor

Monaco-based, tabbed, with syntax highlighting and a chevron dropdown for jumping between open tabs by name.

Script hub

Search and run community scripts from inside the app instead of copy-pasting from a browser tab.

Key system

One temporary key, valid 24 hours, no account required to generate one. Details below.

Customization

Adjust the editor theme and layout without digging through nested settings menus.

Regular updates

Auris ships updates as Roblox changes, so the download page always points at the current build.

A community, not a ticket queue

Bug reports and requests go through Discord, where they're visible to everyone, not stuck in an inbox.

FAQ

Common questions

Is Auris free?

Yes. Generating a key costs nothing — see the Key System page for how it works.

Do I need an account?

No. You can generate a key and download Auris without registering. An account just gives you a dashboard for your key status and a few extras like resetting your HWID yourself.

How long does a key last?

24 hours from the moment it's generated. After that you'll need a new one — the Key System page shows a live countdown so you're never guessing.

What's an HWID reset, and when do I need one?

Your key is tied to a device fingerprint (HWID) so it can't be shared freely. If you switch computers or reinstall, reset it from your user panel — there's a cooldown between resets to keep it meaningful.

Which Roblox version does this work with?

Whatever version is current on the download page's release notes — check there before reporting a compatibility issue.

Where do I report a bug?

In the Discord server. It's faster than email and other people can confirm they're seeing the same thing.
